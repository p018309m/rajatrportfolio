University Solo Project

Unreal Engine 5.4

December~February 2024(8 Weeks)

Gameplay Programmer

Responsibilities

  • Work on the Player and their basic mechanics.
  • Create a parkour system which includes wall running and ledge climbing.
  • Work on the animations using the Motion Matching System.
  • Implement sound and VFX to enhance game feel.

Project Overview

  • Player has basic movement like walking, strafing, sliding and sprinting. Also player has a roll mechanic if jump is timed perfectly.
  • Wall running and ledge climbing are some advanced parkour features that are accessible to the player.
  • The character uses the Motion Matching Animation system which allows for seamless transitions between states within 1 state.

Future Development

  • Design a tutorial level which showcases each mechanic of the player gradually.
  • Add VFX and Sounds to each action the player makes to give more juiciness to the game.
  • Add minimalistic User Interface giving players hint in levels.
