Tech Designer

  • Bubba's Power Nap

Responsibilities

  • Created the whole character, from its movement to its ability to bounce around.
  • Worked on the User Interface of the Game.
  • Created fun obstacles that the player may use for their own advantage.
  • Created some tutorial prompts that can be placed in the world.
  • Managed the Source Control and versioning on GitHub.

Project Overview

  • The character has a simple mechanic which allows the user to launch it based on a meter. The movement is inside a glass bubble which follows physics.
  • A simple HUD which displays information about the player's ability to launch the glass bubble.
  • Face obstacles within the map and use them to progress ahead with the level. Balls are placed around the map to make sure the player can proceed.
  • A fun interactive credits section which the player can play around with.
