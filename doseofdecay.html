Dose of Decay

University Group Project

Unreal Engine 5.2

March-May 2024(10 Weeks)

Roles

Gameplay Programmer

System Designer

UI Tech Designer

Responsibilities

  • Created the Player Ability System and Abilities.
  • Worked on the User Interface of the Game.
  • Implemented a system that was used in the Tutorial Level.
  • Worked on the scrapped boss's prototype.

Project Overview

  • Worked on the Base Player Abilities like double jumping and dashing. Also worked on abilities such as the Air-Strike, Ignition Blast, Vampirism Rage and Distortion Cloak accounting for their durations and cooldowns.
  • Implemented the Player's HUD with elements such as health bar, weapon information, and abilities' information such as durations and cooldowns.
  • Created a system which could hint players on which actors could be interacted with. Also created the UI which shows weapon and ability information, if directly looking at it.
  • Designed the whole tutorial system which allowed designers to move around and edit prompts for the tutorial. Player HUD would appear based on the prompt type to ease the player into the game.
