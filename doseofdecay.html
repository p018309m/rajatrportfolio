Dose of Decay

Project Type - University/Group

Role - Tech Designer

 

Software - Unreal Engine 5.2 and C++

    

This was my first major collaborative project. I worked with a team of 22 people for 10 weeks as a Tech Designer. I mainly worked on the Player Abilities and the User Interface. I also delved into some of the classes in C++ to make the game more efficient. The system for the tutorial prompts in the tutorial level was also built by me which the designers were easily able to edit values into.
