Project Proteus

University Solo Project

Unreal Engine 5.4

October~February 2024(16 Weeks)

Gameplay Programmer

Responsibilities

  • Write a Dissertation on how multi-character systems work in video games.
  • Create a system with multiple unique characters.
  • Work on a level which showcases each character.
  • Create a questionnaire to get feedback and analyse it.

Project Overview

  • Work on the Dissertation which talks about the research done on other games having a multi-character system. Contains the Research Methodology and the Results of the findings from this project.
  • Implemented a system which allows a seamless transition between multiple characters, each hvaing elemental powers.
  • Created an Interface which gives the player freedom to choose from multiple characters.

Future Development

  • Design a level which showcases each character's unique abilities alongside enemies which allows for the same.
  • Add more User Interface for the player to know about the strengths and weaknesses of each character and a system which supports that.
  • Continue working on the Dissertation talking about the research process and the final conclusion of the project.
