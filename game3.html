Forgiveness Mechanics Project

Project Type - University/Solo

 

Software - Unity and C#

    

This is a University project which had me demonstrate various forgiveness mechanics for a 2D platformer. Coyote Time Jumps, Jump Buffering, Corner Clip, and Bumped Head Correction are just few to name them.

I also had to improve upon my previous project by implementing various methods for game feel. I mainly focused on the audio and visual feedback whenever the player interacted with any object in the world.
